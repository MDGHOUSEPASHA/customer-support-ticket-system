# Customer Support Ticket System

## Overview

This project is a Customer Support Ticket System built using a C# WinForms Desktop Application, ASP.NET Web API, and SQL Server database.

The desktop application communicates with the Web API to manage support tickets.

## Tech Stack

Frontend: C# WinForms
Backend: ASP.NET Web API (.NET)
Database: SQL Server
Communication: JSON over HTTP

## Features

### User

* Login
* Create support ticket
* View ticket list

### Admin

* View all tickets
* Assign tickets
* Update ticket status

## Project Structure

TicketSystem.API – Web API backend
TicketSystem.Desktop – Desktop frontend
Database – SQL Server scripts

## How to Run

1. Import the database script into SQL Server
2. Open the solution in Visual Studio
3. Run the API project
4. Run the Desktop application
5. Login and test ticket creation
